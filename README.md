# Split Keyboard 40%

## About
Custom split staggered ortholinear keyboard! Uses choc v2 switches with a rotary knob. 40% with custom layers.

## Idea
I don't particularly want carpal tunnel, and I've found that my typing habits are kind of terrible, so building a split keyboard would solve both of those issues. This is based off the ZSA voyager for layout, but a bit reduced for 40% layout.
